# Tower Defense Game

A tower defense game built with HTML, CSS, and JavaScript.

## Features

- Multiple tower types with unique abilities
- Three maps: Whispering Woods, Frozen Pass, and Desert Oasis
- Quest system with gem rewards
- Inventory system for tower loadouts
- Support tower with barrier, healing, and combat boost abilities
- Admin panel (username: pc_man950)

## How to Play

1. Select a map from the level selection screen
2. Place towers along the path to defend against zombies
3. Complete waves to progress
4. Earn gold and gems to unlock new towers
5. Complete quests for additional rewards
